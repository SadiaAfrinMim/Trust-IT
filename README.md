# Trust-IT (BD) Ltd Landing Page

## Project Overview
A responsive and visually appealing landing page for Trust-IT (BD) Ltd, demonstrating clean UI design, proper HTML structure, responsive layout, CSS styling skills, and JavaScript interactivity.

## Technologies Used
- **HTML5**: Semantic markup and structure
- **CSS3**: Custom styles with Tailwind CSS framework
- **JavaScript (Vanilla)**: Form validation, theme switching, and interactive features

## Instructions to Run the Project
1. Download or clone the project files
2. Open `index.html` in any modern web browser
3. The page will load with all features functional

## Features Included
- Responsive navigation with mobile hamburger menu
- Hero section with company branding and call-to-action
- Services section with 3 service offerings
- About section with company introduction
- Contact form with JavaScript validation
- Dark/Light mode toggle (Bonus)
- Smooth scrolling and animations (Bonus)
- SEO-friendly structure (Bonus)
- Accessibility improvements (Bonus)
